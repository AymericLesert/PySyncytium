# PySyncytium

## Démarrage de l'API

```
cd PySyncytium\workspace
env\Script\activate
python syncytium.py --name api
```

## Démarrage du front HTML

```
cd PySyncytium\workspace
env\Script\activate
python syncytium.py --name web
```

## Démarrage du serveur WebSocket

```
cd PySyncytium\workspace
env\Script\activate
python syncytium.py --name websocket
```

## Démarrage des tests unitaires

```
cd PySyncytium\workspace
env\Script\activate
python -m unittest -v
```
